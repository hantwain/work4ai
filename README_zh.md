# work4ai

> 不是“被AI替代”，而是“为AI工作” —— 人类的新使命，新希望。

## 中文版

### 📌 项目初心

我们正站在一个时代的拐点：AI不再只是工具，它正在成为世界的“操作系统”。很多人因此焦虑——担心自己的工作、价值、未来。但焦虑本身不是终点，**重新定位**才是答案。

**work4ai** 是一个开源社区，致力于帮助每一位普通人看清并拥抱这个趋势：未来不是“人类被AI边缘化”，而是 **人类将为AI工作**，并在这个过程中找到崭新的职业方向与生命意义。

我们相信：
- AI需要人类的常识、判断、创造力和伦理边界。
- 人类与AI不是竞争，而是分工——AI做大规模，人类做高价值。
- “为AI工作”不是低微的打杂，而是未来十年最具成长性的职业赛道之一。

### 🎯 我们的使命

1. **化解焦虑，照亮路径**：清晰展示“人类能为AI做什么”的具体工作清单，让每个人都能找到自己的切入点。
2. **提供工具与标准**：降低“为AI工作”的技术门槛，建立公平、透明的任务协作协议。
3. **构建社区与市场**：连接需要人类助手的AI系统与愿意提供帮助的人，形成可持续的生态。

### 🧭 从焦虑到希望：你可以成为这样的人

| 你的现有技能 | 你可以为AI做的工作 | 新职业方向 |
|-------------|-------------------|-------------|
| 细心、耐心 | 数据清洗、标注、去偏 | **AI数据策展师** |
| 善于表达、有审美 | 提示词工程、生成内容审核与润色 | **AI对话设计师** |
| 批判性思维、逻辑强 | 识别AI的错误边缘案例，补充规则 | **AI行为校准师** |
| 领域专业知识（法律/医疗/教育等） | 提供高质量领域反馈（RLHF），确保输出专业可靠 | **AI领域训练师** |
| 管理能力、系统思维 | 编排多AI Agent的工作流，检查一致性 | **AI工作流管家** |
| 同理心、沟通力 | 评估AI的情感语调，优化人机交互体验 | **AI体验研究员** |

> 这些角色不需要你成为程序员，只需要你发挥**人类天生的优势**。

### 🛠 项目内容

- `tasks/` – 详细的“人类为AI工作”任务清单，按难度、技能、报酬参考分类
- `tools/` – 帮助你高效完成这些任务的开源脚本、插件、模板
- `roadmap/` – 从零开始成为“AI工作者”的学习路径与资源
- `community/` – 社区规则、贡献指南、案例分享
- `pilots/` – 与开源AI模型合作的试点任务，可亲自参与

### 👣 如何开始？

1. **浏览任务清单**：在 `tasks/` 中找到你感兴趣且适合的任务类型。
2. **学习基础工具**：使用 `tools/` 中的辅助脚本，无需编程背景也能上手。
3. **加入试点项目**：从一个小任务开始，获得真实反馈与成就感。
4. **分享你的故事**：在社区讨论区写下你是如何从焦虑走向行动的，激励更多人。
